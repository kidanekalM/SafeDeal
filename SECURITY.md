Reporting a Vulnerability

We take security issues seriously and appreciate your efforts to responsibly disclose vulnerabilities. If you discover a security vulnerability in this project:

    Where to report: Please submit details via our official security reporting channel (e.g., GitHub Security Advisories or email to the project maintainers).

    What to include: Provide a clear description of the vulnerability, steps to reproduce, and any potential impact.
